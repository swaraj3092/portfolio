## 🛠️ Installation & Setup

1.  **Clone the terminal**:
    ```bash
    git clone https://github.com/swaraj3092/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Environment Setup**:
    Create a `.env` file and add your Web3Forms key:
    ```env
    VITE_WEB3FORMS_ACCESS_KEY=your_key_here
    ```

4.  **Launch the Host**:
    ```bash
    npm run dev
    ```
